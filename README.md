# Employee_Salary_Prediction
# Employee Salary Prediction Using Machine Learning

## 📌 Project Overview

This project predicts employee salaries based on their years of experience using Machine Learning algorithms. The goal is to analyze the relationship between experience and salary and compare different machine learning models to determine the most accurate predictor.

---

## 🎯 Objective

To build a machine learning model that can predict an employee's salary based on years of experience.

---

## 📂 Dataset

The dataset contains two columns:

| Column Name | Description |
|------------|-------------|
| YearsExperience | Number of years of work experience |
| Salary | Employee salary |

Dataset File: `Salary_Data.csv`

---

## 🤖 Machine Learning Models Used

### 1. Linear Regression
A regression algorithm used to predict continuous values.

### 2. Decision Tree Regressor
A tree-based machine learning algorithm for regression tasks.

### 3. Random Forest Regressor
An ensemble learning algorithm that combines multiple decision trees.
